1 Introducción

El centro educativo psicopedagógico CROA, es un centro multidisciplinar con metodología propia donde conviven todos los agentes educativos de una forma amena y eficaz.

En CROA trabajan psicología clínica, atención a las necesidades específicas del apoyo educativo, logopedia y metodología de estudio. Cuentan con 6 profesionales que son los encargados de atender a los niños. Podemos encontrar logopedas, psicólogo clínico, psicólogo educativo, pedagogo y profesores.

CROA es un lugar divertido, para ser y estar. Trabaja con niños de entre 3 años hasta edad sin límite. Los niños trabajan de manera individual según sus necesidades.

1.1 Alcance del proyecto

Queremos alcanzar algunos aspectos que nos ha propuesto CROA y son:

Comunicación empleado-cliente

Poder realizar pagos online para mayor comodidad y buen seguimiento del pago del cliente

Información sobre actividades que se realizan en el centro

Poder visualizar los espacios libres en las aulas para poder cuadrar el horario

Recogida de información por parte de la empresa de forma online para mejorar la organización y evitar que esa información quede dispersa

OBJN-001 Comunicación empleado-cliente
Version 1.0 (2024-10-12)
Description Chat interno donde haya comunicación entre la empresa y los usuarios, con función de videollamada.
Importance High
Priority High

Version 1.0 (2024-10-12)
Description Pagos de manera online para que pueda ser realizado a través del móvil y no haya que desplazarse hasta la empresa. Se recibirá un aviso cuando la fecha de pago se aproxime.

Priority High

3 Descripción de la situación actual

3.1 Pros y contras de la situación actual

3.1.1 Fortalezas de la situación actual

STRG-001 Bajos costes
Version 1.0 (2024-10-11)
Description Las herramientas software que usan son muy básicas y apenas supone un coste para la empresa.
STRG-002 Facilidad de uso
Version 1.0 (2024-10-11)
Description Es fácil utilizar a un nivel básico y sin mucha experiencia herramientas como Excel y WhatsApp.
STRG-003 Strength
Version 1.0 (2024-10-11)
Description Al usar WhatsApp, puedes comunicarte de forma sencilla con la empresa de manera inmediata.
STRG-004 Web Básica
Version 1.0 (2024-10-11)
Description Permite a clientes interesados poder encontrar un mínimo de información de la empresa dentro de la página web.

3.1.2 Debilidades de la situación actual

WEAK-001 Riesgos de Datos
Version 1.0 (2024-10-11)
Description WhatsApp y Excel no son herramientas diseñadas para manejar datos sensibles, como direcciones, correos o DNI.
WEAK-002 Eficiencia
Version 1.0 (2024-10-11)
Description Usar Excel para datos internos de la empresa puede ser un proceso largo y además se pueden generar errores al introducir datos.
WEAK-003 Escalabilidad
Version 1.0 (2024-10-11)
Description A medida que la empresa crezca tanto el manejo de datos a través de Excel como la comunicación a través de WhatsApp se volverá muy ineficiente.
WEAK-004 Seguimiento de la evolcuión de la empresa
Version 1.0 (2024-10-11)
Description Con un software tan limitado, no se pueden realizar métricas de forma eficiente para ver como va evolucionando la empresa.
WEAK-005 Dificultad en el seguimiento del paciente
Version 1.0 (2024-10-11)
Description Es difícil hacer un seguimiento del paciente sin informes o plantillas específicos para poder gestionarlos y ver la evolución del paciente.
WEAK-006 Dificultad de gestión de inventario
Version 1.0 (2024-10-11)
Description Es dificil gestionar el inventario sin un software específico, pues puede haber fallos o duplicidades que alteren los datos.

3.3 Entorno tecnológico actual

3.3.1 Descripción del entorno de hardware actual

El hardware utilizado en el centro incluye equipos informáticos básicos como ordenadores personales y dispositivos móviles (smartphones, tablets) utilizados por el personal administrativo y los profesores para gestionar información de alumnos, clases y pagos. Estos dispositivos están conectados a través de redes Wi-Fi convencionales. No se cuenta con servidores locales ni sistemas de almacenamiento dedicados. La gestión de información se complementa con el uso de medios físicos como libretas, que son utilizadas para algunas tareas administrativas.

3.3.2 Descripción del entorno de software actual

El entorno de software en el centro C.R.O.A. se basa en herramientas simples y no especializadas, siendo Microsoft Excel la principal, utilizada para el control de los alumnos, la gestión de pagos, y la organización de horarios y aforos, todo de manera manual. Para la comunicación con los padres y la administración de avisos, recordatorios de pago y firma de documentos, el personal utiliza WhatsApp, lo que implica una gestión personalizada y reiterativa. Además, la promoción y difusión de actividades se realizan a través de la cuenta de Instagram, donde se publican videos de las actividades con los niños. Si bien estas herramientas permiten el manejo básico de las operaciones, la información está dispersa, lo que dificulta la centralización y acceso eficiente a los datos necesarios para la toma de decisiones y la administración del centro.
